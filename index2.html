hyc 常见问题与解决方法

以下内容适用于 hyc 在 macOS / Windows 安装或运行时出现的常见报错。
问题 1：macOS 提示“Apple 无法验证此应用是否包含恶意软件”

解决方法

方法一：在系统设置中允许打开
  1. 打开 系统设置 → 隐私与安全性
  2. 向下滚动到 安全性 区域
  3. 你会看到关于该应用的拦截提示
  4. 点击 “仍要打开”
方法二：通过右键菜单强制打开
  1. 在 Finder 中找到 hyc 应用
  2. 右键点击 应用图标
  3. 选择 “打开”
  4. 在弹窗中再次点击 “打开”

⚠️ 直接双击应用通常不会出现“打开”选项，必须使用右键菜单。

问题 2：macOS 提示“hyc_v1.4.6 已损坏，无法打开。你应该将它移到废纸篓”
解决方法：将应用拖入终端解除限制
  1. 打开 Finder，找到 hyc_v1.3.7
  2. 打开 启动台 → 其他 → 终端
  3. 在终端中输入以下命令（先不要按回车）：
    sudo xattr -rd com.apple.quarantine
  4. 直接将 hyc 应用拖到终端窗口中，系统会自动补全路径
  5. 按 回车，输入电脑开机密码（输入时不显示，属正常）
  6. 再次打开应用即可

说明：这是 macOS 的安全机制导致，不会损坏应用或电脑。

问题 3：Windows 下无法使用键盘移动角色
解决方法：切换为英文输入法
  1. 确认系统中已安装 英文输入法
  2. 按 Shift 或 Win + 空格 切换为英文

注意：游戏内移动必须处于英文输入法状态。
